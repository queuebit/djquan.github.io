Octopress

Well, the switch to Octopress seems pretty complete. Here are just a few of my thoughts on it.

Give octopress a try. It’s easy to set up. It’s easy to post content.

  • Octopress makes it incredibly easy to write and publish from the command line.
  • Everything about an octopress blog will be stored locally.
  • This gives freedom to switch publishing platforms or web hosts and a local backup of posts(with version control!).
  • Octopress is made for coders. Inserting code snippets is incredibly easy. There are also a large number of plugins to extend functionality.

I will miss some aspects of tumblr.

  • Tumblr is an incredible way to make connections and really is social blogging at its best.
  • Tumblr has apps to make mobile publishing simple.
