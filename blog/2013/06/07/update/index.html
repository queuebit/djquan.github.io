Design

I’ve started to dive into some more design resources.

Code School’s Journey into Mobile is a pretty good introduction to responsive design. Responsive design is designing your site so that it looks good in varying resolutions and devices. It is extremely important with the variety of ways people can visit a site. It involves using percentages or ems (relative units in css). Also, it involves finding “break points”, resolutions which the site design changes. All in all, it was eye opening at the amount of thought and work that it takes to make a site responsive; especially since tools like Twitter’s Bootstrap can hide all the magic of responsive design.
